    Install Textext Plugin for Inkscape

    YONG

    2016-04-25 21:45

    Comments

    Textext官网(http://pav.iki.fi/software/textext/)以及作者本人的bitbucket很久没有更新了(https://bitbucket.org/pv/textext/src). 针对Inkscape 0.91版本, 最新的textext资源在这里(https://bitbucket.org/pitgarbe/textext/src)

    Linux下可以直接下载 textext-0.5-linux.zip 安装包, 里面有说明文档, 对于windows, 其pdf说明文档该页面. 对于Linux/Win, 安装文档中要求的其它部件. 之后:

    Linux: 将压缩包里的 textext.py 文件替换成本页面最近更新的 textext.py, 再执行 python setup.py

    Win: 使用exe安装之后, 将 textext.py 替换成本页面最近更新的 textext.py 文件.

    Done.
