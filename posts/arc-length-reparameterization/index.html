    Arc Length Reparameterization

    YONG

    2016-06-28 10:04

    Comments

    reference: https://services.math.duke.edu/~wka/math103/curvature.pdf

    Suppose \(I\) is an interval and \(\mathbf{r} : I \mapsto \mathbb{R}^n\) is a curve in \(\mathbb{R}^n\) whose speed is never zero. Suppose \(t_0 \in I\) and let

    \begin{equation*} \sigma (t) = \int_{t_0}^t \| \vec{r}' (\tau) \| d\tau \end{equation*}

    Given a vector function \(\vec{r}(t)\), its arc length from \(t = a\) to \(t = b\) is

    \begin{equation*} L = \int_a^b \| \vec{r}' (t)\| dt \end{equation*}

    Replacing \(b\) with \(u\), the arc length function is

    \begin{equation*} s(u) = \int_a^u \| \vec{r}' (\tau) \| d\tau \end{equation*}
