    Atom Markdown Settings

    YONG

    2016-04-25 12:22

    Comments

    Markdown Packages:

    • language-markdown
    • markdown-preview-plus
    • markdown-scroll-sync

    Note:

    1. After installing markdown-preview-plus, the built-in package markdown-preview should be disabled.
    2. Disable built-in package whitespace: since it will remove trailing whitespaces automatically, while two whitespaces are necessary in markdown mode. Though you can disable this feature in the package settings, it is strongly recommended to disable the package because it also affects the performance of Atom in markdown mode.
