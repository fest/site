    LyX Settins on Windows

    YONG

    2016-04-25 12:22

    Comments

    Main settings

    Refer to http://andnot.farbox.com/post/ke-yan-bi-ji/lyxzhong-wen-pei-zhi-tips

    Document->Settings…

    • ->Document Class->Document class: Chinese Article (CTeX), ->Custom: UTF8
    • ->Language->Encoding->Other: Unicode (XeTeX) (utf8)
    • ->Output->Output Format->Default Output Format: PDF(XeTeX)
    • ->LaTeX Preamble: \DeclareRobustCommand\nobreakspace{\leavevmode\nobreak\ } % redefine nobreakspace to avoid XeTeX errors
    • ->PDF Properties: check Use Hyperrref Support, ->Bookmarks: check Generate Bookmarks (ToC), Numbered bookmarks, Open bookmark tree

    To aligh the section title in CTeX

    Document->Settings…->LaTeX Preamble:

    \makeatletter

\g@addto@macro{\CTEX@section@format}{\raggedright}

\makeatother
