    • Uniform Sampling

      • LUT_sameSeg.h5: generated by lookupTableEven()

        • lookupTableEven(num::Int64, theta_max, sigma, unitLen)

          Input:

          • num: number of total segments \(N\). A larger \(N\) produces longer curve.

          • theta_max: maximum winding angle of each segment. Actually it is the maximum difference of the winding angle of each segment.

            \begin{equation*} \Delta \theta = \theta(s)-\theta_0=\kappa_0 s + \frac{\sigma s^2}{2} \end{equation*}

            Then, theta_max is \(\Delta \theta_{\max}\). It is not directly used in the function but as a criterion for checking purpose. We set theta_max=0.5*pi by default.

            明显，因为 \(s\) 是个定值，而 \(\sigma\) 也是定值（设为1），\(\Delta \theta_{\max}\) 出现在 \(\kappa_0\) 取最大时，即最末一段。最后一段的起始曲率由下式得到, 其中 \(N\) 是总的段数。

            \begin{equation*} \kappa_{0,N} = \sigma (N-1) s \end{equation*}

            代入到上面的 \(\Delta \theta\) 表达式，得到：

            \begin{equation*} (N-1) \sigma s^2 + \frac{\sigma s^2}{2} \le \Delta \theta_{\max}=\frac{\pi}{2} \end{equation*}

            即，当 \(\sigma=1\) 时, \(N\) 与 \(s\) 要符合上述关系式。

          • sigma: sharpness \(\sigma\)

          • unitLen: length of each small segment \(s\) (they have the same unitLen since they are uniformly sampled)

          Output inside LUT_sameSeg.h5:

          • umax: an array of maximum lengths of all segments
          • utotal: an array of total lengths of all previous segments (not including current one)
          • coeff: an array of bezier coefficients
