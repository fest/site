    Atom LaTeX Settings

    YONG

    2016-04-25 12:21

    Comments

    Packages:

    1. Install TeX Live (https://www.tug.org/texlive/)
    2. Install Atom
    1. Install Atom Packages
      • latex (compile latex)
      • language-lax (syntax highlighting)
      • latexer (autocmpletion)
      • latextools (more support)
      • minimap (preview of code)
      • open-recent (open recent files)
      • pdf-view (builtin pdf viewer)
      • file-icons
      • autocomplete-paths
      • spell-check
    2. Package Settings:
      • latex:
        • Tex Path: C:\texlive\2015\bin\win32
        • Builder: latexmk
        • Always Open Result in Atom: Yes
        • (Other settings: keep default)
      • spell-check:
        • Grammars: text.tex
      • pdf-view:
        • Fit to width on open: Yes

    Important shortcuts:

    • Alt+Shift+S: search for latex snippet
    • Alt+Ctrl+B: compile
    • Alt+Ctrl+S: sync
    • Alt+Ctrl+C: clear
