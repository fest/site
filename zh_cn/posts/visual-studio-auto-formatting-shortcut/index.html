Visual Studio Auto Formatting Shortcut

YONG

2016-04-26 01:04

Comments

Visual Studio auto format (C++) shortcut:

  • To format a selection: CTRL + K + F
  • To format a document: CTRL + K + D

See more in Edit -> Advanced menu
